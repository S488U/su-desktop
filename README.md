# SU Dunite

### To run the app:

Git clone:

```
git clone https://github.com/S488U/su-desktop.git
```

Install the dependencies:

```
npm i
```

To run the app:

```
npx electron .
```

## Install the Desktop App Set up file:  [Download](https://dunite.tech/app/dunite.exe)

## To build the app

```
npm run build
```
